函数式编程（Functional Programming）把函数作为基本运算单元，函数可以作为变量，可以作为参数，可以作为返回值。
历史上研究函数式编程的理论是Lambda演算，因此把支持函数式编程的编码风格称为Lambda表达式，也称为闭包。
适当的使用Lambda表达式，可以是代码变得更加简洁紧凑。

3.3 引用外层局部变量

在lambda表达式中，只能引用外层的有final标记的局部变量。如果要引用没有final标记的外层变量，那么这个变量必须在第一次赋值后就没有被修改过，也就是具有隐含的final性质。
